# Eval Framework

This directory contains tools for blind evaluation of engines built by the skill.

## Files

- `compare-outputs.mjs` — Comparator script that tests a candidate engine against a control baseline
- `BLIND_TEST_INSTRUCTIONS.md` — Instructions for the blind test evaluator
- `control-baseline.json` — **Not included in public repo.** Generated by the evaluator from their reference implementation.

## How to Use

1. Build your reference engines (the "answer key")
2. Run `generate-control.mjs` to produce `control-baseline.json`
3. Give a fresh Claude Code instance the Excel files + this skill + the blind test instructions
4. They build engines using the skill
5. Run `compare-outputs.mjs <candidate-dir>` to score their output
6. Iterate on the skill until blind tests achieve ≥95%
